## Docker Angular

Build docker image:
```
cd ./app
docker build . -t my-app
```

Run docker container:
```
sudo docker run -d -p 80:80 my-app
```

Access to angular application:
http://localhost:80

## Angular

Install depedencies:
```
cd ./app
npm intall
```

Launch application:
```
ng serve
```

Access to angular application:
http://localhost:4200

## Database (Node TS)

Launch database
```
npm start
```

Access to database:
http://localhost:3000
